## v1.4.0

- 按功能拆分 JavaScript 和 CSS，保留现有页面、动画、响应式布局及交互行为。
- 使用 Hugo Pipes 合并、压缩并指纹化主题资源，浏览器仍只加载一个主样式和一个主脚本。
- 增加 GitHub Actions 和 Playwright 回归测试，覆盖主要页面、主题模式、导航、Dock、目录及五种常用视口。
- 完善示例站构建检查、长文章回归页面和本地测试命令。
- 统一 README、示例配置、Hugo Module 和 Go 版本要求，并补充 About 页访客信息说明。
- 补充 Inter 与 Noto Serif SC 的来源、署名和 SIL Open Font License 1.1 文本。

## v1.3.4

1. 保留文章特色字体，TTF 无损转为 WOFF2。字形、字符映射完全一致，体积从 29.56 MB 降至 11.57 MB，减少约 60.9%
2. 首页 Canvas 改为时间驱动，适配 60/120Hz；缓存颜色与渐变，预计算字符行样式，移除每帧遮罩和多余对象分配。
3. 长文章目录改为坐标缓存加二分查找，只在字体、图片或尺寸变化时重新测量。
4. Dock 已删除时间与 IP 地区功能；移动端保持 68–72px 厚度，320px 下也没有重叠。
5. About 页 IP 和位置完全独立，只进入 About 时请求，并加入会话缓存、超时和正则安全处理。
6. 文章列表和分类/标签页加入响应式分页。
7. 修复空目录仍显示侧栏的问题，同时避免发布日期与更新时间重复。
8. Markdown 页面包图片自动生成四档 WebP，保留原图回退，并启用原生懒加载。
9. 修复“减少动态效果”模式下 Dock 合并状态与视觉状态不一致。
10. 删除失效的阅读进度监听、旧图片观察器、废弃 Dock 类和从未触发的事件监听。
11. PC 三合一状态下，品牌、菜单、模式区统一使用 `999px` 胶囊圆角，左右端视觉一致
12. 移动端深浅模式均去除内部底色、边框、阴影和高光，只保留最外层 Dock 胶囊轮廓

## v1.3.2

## v1.3.1

1、分散的版本号统一收拢，现在版本号只在一个地方维护：theme-asset-version.html，新增了一个统一处理入口：theme-asset-url.html，它负责判断：如果资源是主题内置默认资源，就自动加 `?v=...`；如果是用户自定义资源，就只正常解析 URL，不加版本号。已接入这些默认资源：

- 默认 logo / 深色 logo
- 默认 favicon / 深色 favicon
- 默认头像
- 默认文章封面
- 默认 OG 图

2、清掉了之前散落在这些文件里的版本号逻辑：

* header.html
* baseof.html
* index.html
* list.html
* about.html

3、重构三合一 Dock 的交互结构：合并态使用统一外壳，三个内容区只负责位置变化，背景和轮廓由共享 Dock 呈现。
