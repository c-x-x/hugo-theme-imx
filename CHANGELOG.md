# 变更记录

本项目遵循语义化版本。

## 0.1.0 - 2026-06-22

### 新增

- 使用 Hugo Module 分发主题。
- 中文示例站点和完整配置说明。
- 默认头像、文章封面、OG 分享图和 favicon。
- 可配置导航、社交链接、默认图片和 Giscus。
- 浅色、深色和自动三态主题模式。
- 东八区 08:00/18:00 自动切换。
- 响应式导航、全文搜索、文章目录和代码复制。
- 导航滑块使用弹簧物理实现速度拉伸、方向回弹与精确收束。
- 滑块增加双层玻璃、背景模糊、动态高光和虹彩折射边缘。

### 改进

- 导航活动状态支持自定义菜单和子目录部署。
- 搜索索引使用站点相对路径。
- 移除 Google Fonts 外链依赖。
- 增加键盘焦点样式和减少动画支持。
